Verify is a snapshot testing library for .NET. Instead of hand-writing assertions for every field of a complex object, you call `Verify()` and the result is serialized to a file. On the next run, Verify compares the output to the stored snapshot — the test passes if they match, and shows a diff if they don't.

## How it works

<Steps>
  <Step title="Call Verify() in your test">
    Pass any object, string, stream, file, or JSON to `Verify()`. The result is serialized to a `.received.` file.
  </Step>
  <Step title="Review the diff">
    On first run the test fails. A diff tool opens showing the received output. Accept it to create the `.verified.` file.
  </Step>
  <Step title="Commit the verified file">
    The `.verified.` file is committed to source control. Future runs compare against it automatically.
  </Step>
  <Step title="Review changes as they happen">
    When your code changes, Verify shows exactly what changed in the snapshot — making code review fast and precise.
  </Step>
</Steps>
